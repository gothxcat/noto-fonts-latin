# Noto Fonts

A minimal, reduced set of the Noto Fonts typeface by Google Fonts under the included [SIL Open Font License (OFL) v1.1](http://scripts.sil.org/OFL).

Includes Latin scripts for the following fonts:

- Noto Sans
- Noto Serif

Variants:

- Regular
- Italic
- Thin
- Thin Italic
- Light
- Light Italic
- Medium
- Medium Italic
- Bold
- Bold Italic
- Black
- Black Italic
